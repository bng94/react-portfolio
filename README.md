# React Portfolio v2

The second version of my personal portfolio website, built with ReactJS. This iteration features a redesigned, more interactive UI compared to its predecessor, along with an additional section dedicated to highlighting noteworthy projects and showcasing accomplishments beyond the featured work.

The site is structured around a clean single-page layout with sections for an introduction, background, work experience, featured projects, and a noteworthy projects archive. Scroll-based animations and a responsive design are handled through custom React hooks and styled-components.

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Built With

- [ReactJS](https://reactjs.org/)
- [Styled Components](https://styled-components.com/)
